| ID | Requerimiento | Descripción | Prioridad |
|----|--------------|------------|----------|

## ⚙️ 2. Requerimientos No Funcionales

| ID | Requerimiento | Descripción |
|----|--------------|------------|
| RNF01 | Rendimiento | Carga rápida (<2s ideal) |
| RNF02 | Accesibilidad | Buen contraste, navegación clara |
| RNF03 | Escalabilidad | Fácil agregar nuevos proyectos |
| RNF04 | Mantenibilidad | Código limpio (componentes reutilizables) |
| RNF05 | UX/UI | Interfaz moderna y profesional |

| RNF08 | Deploy | Despliegue en hosting |

---

## 🏗️ 3. Requerimientos Técnicos

- Framework: Next.js  
- Estilos: Tailwind CSS  
- UI Components: shadcn/ui  
- Animaciones: Framer Motion  
- Deploy: Vercel  
- Control de versiones: Git  

---

## 🎨 4. Requerimientos de Diseño

- Estilo: minimalista moderno  
- Paleta: oscuro + acentos (azul, morado o verde)  
- Tipografía: limpia y legible  
- Espaciado consistente  
- Animaciones suaves (no exageradas)  
- Jerarquía visual clara  

---

## 🧠 5. Requerimientos de Contenido

- 3–6 proyectos bien explicados (no relleno)  
- Tecnologías reales utilizadas  
- Descripción profesional clara  
- Links funcionales (GitHub/demo)  

---

## 📦 Entregables

- Sitio web funcional desplegado  
- Código fuente en repositorio GitHub  
- Portfolio listo para presentación profesional
